Workspace представляет собой среду для разработки.  
Основная задача: упростить и облегчить сборку продуктов.  
WorkSpace предлагает:  
  - продуманная организация процесса разработки.  
  - набор инструментов для комфортной работы.  
  - возможность собирать сразу множество конфигураций, множеством компиляторов.  


Основные идеи:
--------------

- комфорт разработчика.  
- продуманная иерархия каталогов.  
- простые и практичные правила.  
- готовый набор инструментов.  
- автоматизация рутины.
